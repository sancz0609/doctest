DocTest



This application takes a base document as a "BASE" and compares it with the document we need to evaluate.
The parameters used here are the word count of the document. Using the word count of the base document , we determine the range of the word count of the evaluating document. Further , this document takes into account the nouns of the base document and compares it with the nouns of the evaluating document. It then gives out the similar nouns used.
